# Backend - CF

Deploy into Cloudflare Workers

```
$ npm run deploy
```

# Frontend - UI


## Development

```
$ npm install
```

Run a dev server:

```
$ npm run start
```

The check out http://localhost:8000/dev/index.html

Build:

```
$ npm run build
```
